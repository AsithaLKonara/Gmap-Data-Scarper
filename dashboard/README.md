## Installation

Install dashboard dependencies:

```bash
pip install streamlit pandas
```

Or install all requirements:

```bash
pip install -r requirements.txt
```

## Usage

### Start the Dashboard

```bash
streamlit run dashboard/app.py
```

The dashboard will open in your default web browser at `http://localhost:8501`

### Using the Dashboard

1. **Load Data**
   - Enter the path to your CSV file in the sidebar, OR
   - Upload a CSV file directly using the file uploader

2. **Apply Filters**
   - Use the sidebar filters to narrow down your leads
   - Filters can be combined for complex queries
   - See filtered count update in real-time

3. **View Analytics**
   - Charts automatically update based on filters
   - View distribution of leads across different dimensions

4. **Explore Data**
   - Search within the data table
   - Sort by different columns
   - View all lead details

5. **Export**
   - Download filtered results as CSV
   - Export includes all applied filters

## Example Workflows

### Find High-Quality Restaurant Leads in New York

1. Set "Minimum Lead Score" to 70
2. Select "restaurant" in Business Types
3. Select "New York" in Cities
4. View filtered results and export

### Find Active Marketing Agencies

1. Select "marketing" in Business Types
2. Check "Boosted Posts Only"

## Tips

- **Performance**: Large CSV files (>10K rows) may load slowly. Consider filtering first.
- **Search**: Use the search box to quickly find specific leads by name or URL
- **Export**: Always export filtered results to preserve your analysis
- **Refresh**: The dashboard auto-refreshes when you change filters

## Troubleshooting

### Dashboard won't start

```bash
# Check if Streamlit is installed
pip install streamlit pandas

# Try running directly
python -m streamlit run dashboard/app.py
```

### CSV file not loading

- Check file path is correct
- Ensure CSV file exists
- Try uploading file directly instead of using path
- Check CSV file is valid (not corrupted)

### Charts not showing

- Ensure CSV has required columns (Platform, business_type, etc.)
- Check if filters are too restrictive (no data matches)
- Verify data is not all "N/A"

### Performance issues

- Use filters to reduce dataset size
- Consider splitting large CSV files
- Close other applications to free up memory

## Integration

The dashboard works with CSV files generated by the scraper. It automatically detects v2.0+ fields and provides appropriate visualizations.

For programmatic access, use the enrichment API:

```python
from enrichment.enrich_existing import enrich_existing_csv

enrich_existing_csv("input.csv", "output_enriched.csv")
```

Then load the enriched CSV in the dashboard for visualization.
